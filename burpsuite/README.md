### Installation

> Before installation you have to install [nethunter rootless](../nethunter-rootless) and run [kali linux gui desktop](../kali-linux-gui-desktop)

```
nh
```
```
sudo apt install burpsuite
``` 
### Run

```
burpsuite
```

> _Automatically_ appear on the screen
